## Usage

```tsx
import {
  PageHeader,
  PageHeaderBreadcrumb,
  PageHeaderRow,
  PageHeaderTitle,
  PageHeaderDescription,
  PageHeaderActions,
} from '@acronis-platform/ui-react';
```

`PageHeader` is the header region at the top of a page's content (inside the App
Shell's main, not the app-wide [`AppShellHeader`](/layouts/app-shell)). It stacks
an optional breadcrumb, a title row (`PageHeaderRow` — title left, actions right),
and an optional description. Mapped to the
[Page Header Figma](https://www.figma.com/design/lrU3ydIyvPYQNE6ixdsKtJ/shadcn-uikit?node-id=2850-701).

## Parts

| Export                  | Element  | Purpose                          |
| ----------------------- | -------- | -------------------------------- |
| `PageHeader`            | `div`    | The banner region.               |
| `PageHeaderBreadcrumb`  | `nav`    | Optional breadcrumb (muted).     |
| `PageHeaderRow`         | `div`    | Title + actions row.             |
| `PageHeaderTitle`       | `h1`     | The page title.                  |
| `PageHeaderDescription` | `p`      | Optional muted text.             |
| `PageHeaderActions`     | `div`    | Trailing action buttons.         |

All parts accept their native element attributes plus `className`.
